<h1 align="center">
Wonnaway | Хочу уехать
</h1>

<p align="center"><strong>Бот для поиска авиабилетов</strong></p>

### Возможности

- Вывод популярных направлений по поиску билетов
- Вывод популярных направлений у авиакомпании
- Подбор дешевых авиабилетов по датам
- Подбор дешевых авиабилетов по месту назначения
- Курс валют

### Установка

```bash
git clone https://gitlab.com/zarip0v/wonnaway.git
cd wonnaway
pip3 install -r requirements.txt
```
